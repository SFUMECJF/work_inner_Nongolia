## 0.2 编制
通俗来讲，编制就是高人一等的护身符，就是你到底属不属于某个组织的标志，是指组织机构的设置及其人员数量的定额和职务的分配，由财政拨款的编制数额由各级机构编制部门制定，财政部门据此拨款。

有行政编制（`公务员`）、事业编制（`老师`、医生、`科研人员`等）、国企编制比如`银行编制`（正式工,信息科技岗）等。

## 1.1 教师分类
### 高中教师：
在比较好的一些高中，只有本硕专业一致，确实是所教的专业，才能应聘。工科专业，只能试试计算机老师。这里其实也有个问题可以考虑：
主科目老师：
- 假期，编制
- 辛苦，早晚自习，工资不一定高。
计算机老师：
- 假期，编制
- 没学生成绩压力
- 没课上，会做杂活，搬桌子，财务表格
- 升不了职称

信息学竞赛老师：
- 开的学校少
- 具杭州的前辈说：工资低，事情多
所以是否要做高中老师呢？
### 初中教师
优点：
- 有成就感
如果做初中教师，是做计算机老师呢？还是物理老师呢？在初中，计算机毕竟要作为一个考试科目，还是相对比较重要的。那么，也有编制，不过可能上升不太容易 。但是，如果有更自由的时间划水，是不是也可以？有必要一定要做物理这样的主科目老师吗？

### 小学教师
优点：
- 教学内容简单
- 没有早晚自习
缺点：
- 没有成就感。工作内容简单，1~3年级，10以内的加减法

# 2 公务员
负责统筹管理经济社会秩序和国家公共资源，维护国家法律规定，贯彻执行相关义务的公职人员。
基本要求：18周岁~35周岁；具有大专以上文化程度；


电力，烟草

## 2.1 选调
选调，只有应届生才可以，把握机会。
也是公务员，但是，是由**各省区市党委组织部**有计划地从高等院校选调的**品学兼优**的应届大学**本科**及其以上的毕业生的简称。
培养目标：培养方向主要是党政**领导**干部后备人选和县级以上党政机关**高素质**的工作人员人选。
## 2.2 国考
中央机关及其直属机构公务员招录考试
3类试卷：中央机关及其省级直属机构综合管理类职位（副省级）、市（地）级及以下直属机构综合管理类职位、行政执法类。
笔试时间：11月下旬，最后一个周日
内容：上午行测、下午申论

## 2.3 选调生考试

**中央选调生、各省区市选调生**
央选：参加国考；各省区市：自行组织考试
时间：从十月底到次年一月
内容：央选：同国考副省级试卷各省区市：行测、申论一张卷

## 2.4 省考
省、市（州）、县（区）政府及直属机关和地方党群机关的公务员招录考试
时间：不固定，一般在国考后
内容：上午行测、下午申论
## 2.5 中央选调生
纸质版的招录通知及岗位信息，不可拍照上网
高校范围及人数：31所985院校，每个学校200人，中央财经大学、中国政法大学特定专业各50人
报名条件：中共党员；获得过校级以上“三好学生”、“优秀学生干部”等荣誉；……
选拔方式：学校给学院分配名额，学院内部综合考评后推荐人选
