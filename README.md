## 0.2 编制
通俗来讲，编制就是高人一等的护身符，就是你到底属不属于某个组织的标志，是指组织机构的设置及其人员数量的定额和职务的分配，由财政拨款的编制数额由各级机构编制部门制定，财政部门据此拨款。

有行政编制（`公务员`）、事业编制（`老师`、医生、`科研人员`等）、国企编制比如`银行编制`（正式工,信息科技岗）等。
[介绍](https://zhuanlan.zhihu.com/p/136681362)
电网等单位需要考编

## 1.1 教师分类
### 高中教师：
在比较好的一些高中，只有本硕专业一致，确实是所教的专业，才能应聘。工科专业，只能试试计算机老师。这里其实也有个问题可以考虑：
主科目老师：
- 假期，编制
- 辛苦，早晚自习，工资不一定高。
计算机老师：
- 假期，编制
- 没学生成绩压力
- 没课上，会做杂活，搬桌子，财务表格
- 升不了职称

信息学竞赛老师：
- 开的学校少
- 具杭州的前辈说：工资低，事情多
所以是否要做高中老师呢？
### 初中教师
优点：
- 有成就感
如果做初中教师，是做计算机老师呢？还是物理老师呢？在初中，计算机毕竟要作为一个考试科目，还是相对比较重要的。那么，也有编制，不过可能上升不太容易 。但是，如果有更自由的时间划水，是不是也可以？有必要一定要做物理这样的主科目老师吗？

### 小学教师
优点：
- 教学内容简单
- 没有早晚自习
缺点：
- 没有成就感。工作内容简单，1~3年级，10以内的加减法
